# Health Guard ML

Heart disease prediction using XGBoost with SHAP model explainability.

## Features
- XGBoost classifier trained on heart disease dataset
- Model evaluation with precision, recall, F1-score
- SHAP-based feature importance visualization
- Clean, modular Python code ready for extension

## How to Run
```
pip install -r requirements.txt
python main.py
```

## Tech Stack
- XGBoost (gradient boosting)
- scikit-learn (metrics, splitting)
- SHAP (feature importance)
- Matplotlib (visualization)
- Pandas (data handling)
